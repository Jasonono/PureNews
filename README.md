### PureNews
一个简洁的新闻资讯类APP，主要包括新闻、视频、天气三大模块。[点击下载APK][5]

![新闻][1] ![视频][2] ![天气][3] ![我的][4]

#### 涉及到的知识点有：
1.使用Material	Design设计规范；
2.RecycleView+CardView的使用；
3.使用SwipeRefreshLayout来进行下拉刷新；
4.使用BaseRecyclerViewAdapterHelper进行Adapter的优化编写；
5.Glide进行图片加载；
6.使用OkGo网络请求框架；
7.ButterKnife的使用；
8.高德地图定位；
9.TabLayout的使用；
10.CoordinatorLayout+AppBarLayout实现上滑隐藏；
11.WebView的使用，自定义滚动条；
12.使用JieCaoVideoPlayer第三方视频播放控件；
13.AHBottomNavigation底部导航的使用；
